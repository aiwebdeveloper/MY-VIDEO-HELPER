# MY VIDEO HELPER - Professional Video Editor & Management System

A high-performance, browser-based video editing platform with built-in Master Admin Control, Security Hardening, and Subscription Management.

## 🚀 Key Features

### 🎬 Video Editor Core
- **Pro Editing Suite:** Full timeline editor, chroma key (green screen), filters, and transitions.
- **AI Background Tech:** Real-time selfie segmentation.
- **Asset Store:** One-click copyright-free music and effects.

### 🛡️ Security Master (Kinetic Shield)
- **Anti-Bot Certification:** Math-based captcha challenge for signups.
- **Trial Protection:** IP-based tracking to prevent multiple trials on the same device.
- **Hardened Passwords:** Mandatory complex password requirements.
- **Anti-Tamper Shield:** Self-destruct logic for pirate/cracked versions.
- **Deep Watermarks:** Obfuscated, persistent watermarks on exported content.

### 💼 Admin Master Panel
- **User Management:** Extend, revoke, or upgrade user access.
- **Fee Plan Manager:** Create custom Monthly, Yearly, or **Unlimited Lifetime** plans.
- **Discount Coupons:** Toggle percentage or fixed PKR promo codes.
- **Security Logs:** Live tracking of failed logins and hacking attempts with IP logging.

### 👤 Client Dashboard
- **Profile Info:** View Plan type, Join Date, and Expiry Date.
- **Age Restricted:** Mandatory 18+ age validation.
- **Agreement System:** Formal Terms of Service acceptance.
